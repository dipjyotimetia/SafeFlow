# Backup and Sync Guide

## Local backup (recommended baseline)

1. Go to **Settings**.
2. Click **Export Backup**.
3. Store the exported file in a secure location (encrypted storage preferred).
4. Repeat on a schedule (weekly minimum).

## Restore from backup

1. Go to **Settings**.
2. Click **Import Backup**.
3. Choose a SafeFlow backup file.
4. Verify balances and recent transactions after restore.

## Encrypted cloud sync

1. Open **Settings > Cloud Sync**.
2. Choose provider and configure credentials.
3. Set a strong encryption password.
4. Run a sync and check status.

## Password guidance

- Use a unique passphrase (12+ chars).
- Store it in a password manager.
- Do not share it in plaintext channels.

## Recovery checklist

- Confirm provider credentials.
- Confirm encryption password correctness.
- Try importing latest local backup first.
- Check browser storage permissions.
